# 🪑 FurnitureSite

Şık ve modern mobilya tasarımlarını sergilemek, kullanıcılara ilham vermek ve alışveriş deneyimini dijitalleştirmek amacıyla geliştirilmiş bir web projesidir. Kullanıcı dostu arayüzü ve performans odaklı yapısı ile mobilya sektörüne yönelik bir çözüm sunar.

---

## ✨ Özellikler

* **Ürün Vitrini:** Kategorize edilmiş mobilya listeleri (Koltuk takımları, yatak odası, ofis mobilyaları vb.).
* **Detaylı Ürün Sayfaları:** Ürün özelliklerini, boyutlarını ve yüksek çözünürlüklü görsellerini inceleme.
* **Duyarlı (Responsive) Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu arayüz.
* **Kullanıcı Etkileşimi:** Ürünleri filtreleme, arama ve favorilere ekleme (veya sepete ekleme) özellikleri.
* **Modern UI/UX:** Temiz, ferah ve kullanıcıyı yormayan tasarım dili.

## 🚀 Kurulum

Projeyi yerel ortamınızda ayağa kaldırmak için:

1. **Repoyu bilgisayarınıza indirin:**
```bash
git clone https://github.com/EgehanHatipoglu/FurnitureSite.git

```


2. **Proje dizinine gidin:**
```bash
cd FurnitureSite

```


3. **Gerekli bağımlılıkları yükleyin:**
```bash
# Örnek (npm kullanıyorsan):
npm install

```


4. **Uygulamayı başlatın:**
```bash
# Örnek:
npm start

```
